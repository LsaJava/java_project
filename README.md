# 🚚 Exercícios de Lógica - Foco em Logística (Java)

Repositório criado para armazenar os exercícios de Algoritmos e Lógica de Programação do curso de **ADS**.

Os exercícios simulam situações reais do dia a dia de um motorista de caminhão e operações logísticas, utilizando:
- Entrada e Saída de dados (`Scanner`)
- Estruturas de Condição (`if`, `else if`, `else`)
- Operadores Lógicos e Matemáticos

### 🛠️ Tecnologias
- Java JDK 17
- IntelliJ IDEA
